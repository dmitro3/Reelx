# Админ-панель

Веб-интерфейс для управления системой.

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` на основе `.env.example`:

```env
VITE_API_BASE=http://localhost:3001/api/admin-c7ad44cbad762a5da0a4/admin
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Функционал

### Страницы

1. **Login** - Авторизация администратора
2. **Пользователи** - Управление пользователями:
   - Список пользователей с фильтрацией
   - Поиск по ID, нику
   - Бан/разбан пользователей
   - Корректировка баланса (TON/STARS)
   - Просмотр инвентаря

3. **Статистика игр** - Аналитика игр:
   - Общее количество игр
   - Статистика по режимам (Solo/PVP/Upgrade)
   - Заработано (Rake + RTP)
   - Оборот (сумма всех ставок)
   - Фильтры по датам

4. **Транзакции** - История всех операций:
   - Ввод/вывод средств
   - Ставки
   - Фильтры по пользователю, типу, датам
   - Статусы и суммы

5. **Настройки** - Параметры системы:
   - RTP для Solo/Upgrade
   - Rake для PVP
   - История изменений настроек

6. **Промокоды** - Управление промокодами:
   - Создание промокодов
   - Типы: бонус к балансу или при пополнении
   - Валюта: TON или STARS
   - Просмотр использования

## API Endpoints

Все API вызовы идут через сервисы в `src/services/`:

- `authService` - авторизация
- `usersService` - управление пользователями
- `gamesService` - статистика игр
- `transactionsService` - транзакции
- `settingsService` - настройки
- `promocodesService` - промокоды

## Структура проекта

```
src/
  components/     # Переиспользуемые компоненты
  contexts/       # React контексты (Auth)
  pages/          # Страницы приложения
  services/       # API сервисы
  App.tsx         # Главный компонент с роутингом
  main.tsx        # Точка входа
```
